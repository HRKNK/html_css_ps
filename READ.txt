- <h1> - только один заголовок первого уровня на всю страницу, необходим для корректного чтения роботами/скринридером 
        далее: h2, h3

- <p> - параграф - текст/абзац
- <div> - тег группировки элементов (разделитель контента) / блочный элемент
- <span> - позволяет стилизовать часть текста / строчный элемент
- <sup> - надстрочный текст | - <sub> - подстрочный текст (чаще используется в формулах)

- <ul> - Unordered List - неупорядоченный список
- <ol> - Order List - упорядоченный список, который может содержать внутри себя теги <li>

Семантическая разметка – создание осмысленной и логически связанной информационной структуры содержимого (контента)
    - конкретное описание

- <header>, <footer>, <nav> - навигация, <section>, <article>, <aside> - боковая панель, <main> - основная часть (содержит в себе секции, статьи и т.п.)
- Селекторный сниппет VSCode: header>nav>ul>li*5
